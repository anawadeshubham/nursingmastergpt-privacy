Nursing MasterGPT — Privacy Policy

Last updated: 29 November 2025

Nursing MasterGPT is an educational AI assistant built to help BSc Nursing students. This privacy policy explains how Google OAuth access is used for the application.

What access the app requests

The application requests read-only Google Drive access (scope: https://www.googleapis.com/auth/drive.readonly) so the app can read files in the developer’s specified Drive folder to generate study content and exam-ready answers. The app does not request write or delete permissions.

Data usage

The app uses OAuth solely to read files from the developer’s Google Drive folder. Files are used to produce study material, summaries, care plans, MCQs and other outputs in chat. No user personal information is collected, stored or shared.

Storage & retention

No user data is stored by any third party as part of this app. OAuth tokens are used only to perform Drive reads during an active session.

Revoking access

Users (including the developer) can revoke this app’s access anytime from their Google Account → Security → Third-party apps with account access.
